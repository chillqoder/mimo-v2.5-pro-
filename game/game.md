Create a single-file HTML game using Three.js that is a 3D low-poly interpretation of the Google Chrome Dinosaur endless runner. The game should feature a procedurally generated stylized environment (e.g., a low-poly desert or prehistoric landscape). The main character, a low-poly T-Rex build from basic shapes, must run forward automatically. Implement jumping mechanics (spacebar input) to avoid obstacles like low-poly cacti or rocks. As the game progresses, the speed should increase and a score should be displayed in the corner. Use a vibrant, stylized color palette with soft lighting to give it a modern aesthetic. The entire code must be contained within one HTML file, including the Three.js library via CDN, and it must handle window resizing.
